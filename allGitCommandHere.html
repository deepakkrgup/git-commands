Git Configuration

command : git config --global user.name "Your GitHub UserName"

command : git config --global user.email "Your GitHub Email"

Verify the Configuration

command : git --global --list

output : Your UserName and Your email

Clone & Status

command : git clone repository-url
For Example : git clone https://github.com.user/repository.git

It used to download(copy) a remote Git Repository to your local computer.

Status

command : git status

it is used to show status of your git repository.
